# Sources for data migration<a name="CHAP_Source"></a>

AWS Database Migration Service \(AWS DMS\) can use many of the most popular data engines as a source for data replication\. The database source can be a self\-managed engine running on anÂ Amazon EC2 instance or an on\-premises database\. Or it can be a data source on an AWS service such as Amazon RDS or Amazon S3\.
